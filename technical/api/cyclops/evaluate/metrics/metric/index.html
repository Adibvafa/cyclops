class MetricCollection(metrics: Union[Metric, Sequence[Metric], Dict[str, Metric]])[source]ÔÉÅ

Bases: Metric

A collection of metrics.

Provides a convenient way to compute multiple metrics at once. It groups metrics that have similar state variables and only updates the state variables once per group, reducing the amount of computation required.

Parameters

metrics (List[Metric]) ‚Äì The list of metrics to collect.
