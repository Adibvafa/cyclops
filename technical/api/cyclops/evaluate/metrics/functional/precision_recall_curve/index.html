Compute precision-recall curve for binary input.

Parameters

  • target (ArrayLike) ‚Äì Binary target values.

  • preds (ArrayLike) ‚Äì Predicted probabilities or output of a decision function. If preds are logits, they will be transformed to probabilities via the sigmoid function.

  • thresholds (int or list of floats or numpy.ndarray of floats, default=None) ‚Äì Thresholds used for computing the precision and recall scores. If int, then the number of thresholds to use. If list or numpy.ndarray, then the thresholds to use. If None, then the thresholds are automatically determined by the unique values in preds.

  • pos_label (int) ‚Äì The label of the positive class.

Returns

  • precision (numpy.ndarray) ‚Äì Precision scores such that element i is the precision of predictions with score >= thresholds[i].

  • recall (numpy.ndarray) ‚Äì Recall scores in descending order.

  • thresholds (numpy.ndarray) ‚Äì Thresholds used for computing the precision and recall scores.

Examples

>>> from cyclops.evaluation.metrics.functional import (
...     binary_precision_recall_curve
... )
>>> target = [0, 0, 1, 1]
>>> preds = [0.1, 0.4, 0.35, 0.8]
>>> precision, recall, thresholds = binary_precision_recall_curve(target,
...     preds, thresholds=5
... )
>>> precision
array([0.5, 0.66666667, 1., 1., 0.]
>>> recall
array([1., 1., 0.5, 0.5, 0.])
>>> thresholds
array([0.1, 0.25 , 0.5, 0.75 , 1.])

Parameters

  • target (ArrayLike) ‚Äì Ground truth (correct) target values.

  • preds (ArrayLike) ‚Äì Estimated probabilities or decision function. If preds is a logit, it will be converted to a probability using the softmax function.

  • thresholds (Union[int, List[float], numpy.ndarray], default=None) ‚Äì Thresholds used for computing the precision and recall scores. If int, then the number of thresholds to use. If list or array, then the thresholds to use. If None, then the thresholds are automatically determined by the unique values in preds.

Returns

  • precision (numpy.ndarray or list of numpy.ndarray) ‚Äì Precision scores where element i is the precision score corresponding to the threshold i. If state is a tuple of the target and predicted probabilities, then precision is a list of arrays, where each array corresponds to the precision scores for a class.

  • recall (numpy.ndarray or list of numpy.ndarray) ‚Äì Recall scores where element i is the recall score corresponding to the threshold i. If state is a tuple of the target and predicted probabilities, then recall is a list of arrays, where each array corresponds to the recall scores for a class.

  • thresholds (numpy.ndarray or list of numpy.ndarray) ‚Äì Thresholds used for computing the precision and recall scores.

  • thresholds (numpy.ndarray or List[numpy.ndarray]) ‚Äì If thresholds is None, then thresholds is a list of arrays, one for each label. Otherwise, thresholds is a single array with shape (len(thresholds,).

Compute the precision-recall curve for different tasks/input types.

Parameters

  • target (ArrayLike) ‚Äì Ground truth (correct) target values.

  • preds (ArrayLike) ‚Äì Estimated probabilities or non-thresholded output of decision function.

  • task (Literal["binary", "multiclass", "multilabel"]) ‚Äì The task for which the precision-recall curve is computed.

  • thresholds (int or list of floats or numpy.ndarray of floats, default=None) ‚Äì Thresholds used for computing the precision and recall scores. If int, then the number of thresholds to use. If list or array, then the thresholds to use. If None, then the thresholds are automatically determined by the sunique values in preds

  • pos_label (int, default=1) ‚Äì The label of the positive class.

  • num_classes (int, optional) ‚Äì The number of classes in the dataset. Required if task is "multiclass".

  • num_labels (int, optional) ‚Äì The number of labels in the dataset. Required if task is "multilabel".

Returns

  • precision (numpy.ndarray) ‚Äì The precision scores where precision[i] is the precision score for scores >= thresholds[i]. If task is ‚Äòmulticlass‚Äô or ‚Äòmultilaabel‚Äô, then precision is a list of numpy arrays, where precision[i] is the precision scores for class or label i.

  • recall (numpy.ndarray) ‚Äì The recall scores where recall[i] is the recall score for scores >= thresholds[i]. If task is ‚Äòmulticlass‚Äô or ‚Äòmultilaabel‚Äô, then recall is a list of numpy arrays, where recall[i] is the recall scores for class or label i.

  • thresholds (numpy.ndarray) ‚Äì Thresholds used for computing the precision and recall scores.

Raises

  • ValueError ‚Äì If task is not one of ‚Äòbinary‚Äô, ‚Äòmulticlass‚Äô or ‚Äòmultilabel‚Äô.

  • AssertionError ‚Äì If task is multiclass and num_classes is not provided.

  • AssertionError ‚Äì If task is multilabel and num_labels is not provided.

Examples

>>> # (binary)
>>> from cyclops.evaluation.metrics.functional import precision_recall_curve
>>> target = [0, 0, 1, 1]
>>> preds = [0.1, 0.4, 0.35, 0.8]
>>> precision, recall, thresholds = precision_recall_curve(target, preds,
...     "binary")
>>> precision
array([0.66666667, 0.5, 1., 1.])
>>> recall

>>> thresholds
