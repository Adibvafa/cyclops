cyclops.evaluate.metrics.functional.stat_scoresïƒ

Functions for computing stat scores for different types of inputs.

The stat scores are the number of true positives, false positives, true negatives, and false negatives. Binary, multiclass and multilabel data are supported, including logits and probabilities.

Compute stat scores for binary, multiclass or multilabel problems.

This function acts as an entry point to the specialized functions for each task.

Parameters

  • target (ArrayLike) â€“ Ground truth.

  • preds (ArrayLike) â€“ Predictions.

  • task (Literal["binary", "multiclass", "multilabel"]) â€“ The task type. Can be either binary, multiclass or multilabel.

  • pos_label (int, default=1) â€“ The positive label to report. Only used for binary tasks.

  • threshold (float, default=0.5) â€“ The threshold to use for binarizing the predictions if logits or probabilities are provided. If logits are provided, a sigmoid function is applied prior to binarization. Used for binary and multilabel tasks.

  • num_classes (int) â€“ The number of classes for the problem. Required for multiclass tasks.

  • classwise (bool, default=True) â€“ Whether to return the stat scores for each class or sum over all classes. Only used for multiclass tasks.

  • top_k (int, default=None) â€“ The number of top predictions to consider when computing the statistics. If None, top_k is set to 1. Used for multiclass and multilabel tasks.

  • num_labels (int) â€“ The number of labels. Only used for multilabel tasks.

  • labelwise (bool, default=False) â€“ Whether to compute the stat scores labelwise. Only used for multilabel tasks.

Returns

scores â€“ The stat scores - true positives, false positives, true negatives, false negatives and support. For binary tasks, the shape is (5,). For multiclass tasks, the shape is (n_classes, 5) if classwise is True, otherwise (5,). For multilabel tasks, the shape is (n_labels, 5) if labelwise is True, otherwise (n_classes, 5).

Return type

numpy.ndarray

Examples

>>> # (binary)
>>> from cyclops.evaluation.metrics.functional import tat_scores
>>> target = [0, 1, 1, 0]
>>> preds = [0, 1, 0, 0]
>>> stat_scores(target, preds, task="binary")
array([1, 0, 2, 1, 2])

>>> # (multiclass)
>>> from cyclops.evaluation.metrics.functional import multiclass_stat_scores
>>> target = [0, 1, 2, 2, 2]
>>> preds = [0, 2, 1, 2, 0]
>>> stat_scores(target, preds, task="multiclass", num_classes=3)
array([[1, 1, 3, 0, 1],
        [0, 1, 3, 1, 1],
        [1, 1, 1, 2, 3]])

>>> # (multilabel)
>>> from cyclops.evaluation.metrics.functional import stat_scores
