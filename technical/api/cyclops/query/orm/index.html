query_batch_conditions(query: cyclops.query.util.TableTypes, id_col: str, batch_size: int) → List[Union[sqlalchemy.sql.elements.BinaryExpression, sqlalchemy.sql.elements.BooleanClauseList]][source]ÔÉÅ

Return a list of WHERE conditions to segment a query into batches.

Batches are created via SQL windowing, based on segmenting the values in a given column, such as an ID column, into intervals.

Requires a database that supports window functions.

Parameters

  • column (sqlalchemy.sql.schema.Column) ‚Äì The column over which to create the interval ranges and conditions.

  • window_size (int) ‚Äì The range of the interval to create.

Returns

  • list of sqlalchemy.sql.elements.BinaryExpression or

  • sqlalchemy.sql.elements.BooleanClauseList ‚Äì The window conditions on which to filter.
