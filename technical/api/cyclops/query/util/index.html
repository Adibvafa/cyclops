assert_table_has_columns(*args, **kwargs) → Callable[source]

Assert that TableTypes params have the necessary columns.

assert_table_has_columns([“A”, “B”], None) is equivalent to assert_table_has_columns([“A”, “B”]) but may be necessary when wanting to check, assert_table_has_columns([“A”], None, [“C”])

Can also check keyword arguments, e.g., optional queries, assert_table_has_columns([“A”], kwarg_table=[“D”])

Parameters

  • *args – Ordered arguments corresponding to the function’s table-type args.

  • **kwargs – Keyword arguments corresponding to the function’s table-type kwargs.

Returns

Decorator function.

Return type
