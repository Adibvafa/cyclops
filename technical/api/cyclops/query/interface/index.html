class QueryInterfaceProcessed[source]ÔÉÅ

An interface dataclass to wrap queries, and run them with post-processing.

A similar dataclass to QueryInterface, where custom post-processing functions on the pandas dataframe returned from the query can be run. However, this prevents the query from being further used, and hence is declared as private attribute.

Parameters

  • database (cyclops.orm.Database) ‚Äì Database object to create ORM, and query data.

  • _query (cyclops.query.util.TableTypes) ‚Äì The query.

  • process_fn (Callable) ‚Äì Process function to apply on the pandas dataframe returned from the query.

  • _run_args (dict) ‚Äì Private dictionary attribute to keep track of arguments passed to run() method.

  • data (pandas.DataFrame) ‚Äì Data returned from executing the query, as Pandas DataFrame.
