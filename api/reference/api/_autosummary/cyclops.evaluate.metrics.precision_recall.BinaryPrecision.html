Compute the precision score for binary classification tasks.

Parameters:

  • pos_label (int, default=1) ‚Äì The label of the positive class.

  • threshold (float, default=0.5) ‚Äì Threshold for deciding the positive class.

  • zero_division (Literal["warn", 0, 1], default="warn") ‚Äì Value to return when there is a zero division. If set to ‚Äúwarn‚Äù, this acts as 0, but warnings are also raised.

Examples

>>> from cyclops.evaluation.metrics import BinaryPrecision
>>> target = [0, 1, 0, 1]
>>> preds = [0, 1, 1, 1]
>>> metric = BinaryPrecision()
>>> metric(target, preds)

>>> metric.reset_state()
>>> target = [[0, 1, 0, 1], [0, 0, 1, 1]]
>>> preds = [[0.1, 0.9, 0.8, 0.2], [0.2, 0.3, 0.6, 0.1]]
>>> for t, p in zip(target, preds):
...     metric.update_state(t, p)
>>> metric.compute()
0.6666666666666666
