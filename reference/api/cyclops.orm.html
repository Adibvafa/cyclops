query_batch_conditions(query, id_col, batch_size)[source]ÔÉÅ

Return a list of WHERE conditions to segment a query into batches.

Batches are created via SQL windowing, based on segmenting the values in a given column, such as an ID column, into intervals.

Requires a database that supports window functions.
