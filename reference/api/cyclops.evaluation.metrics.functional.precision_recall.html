Compute precision score for multiclass classification.

Parameters

  • target (ArrayLike) ‚Äì Ground truth (correct) target values.

  • preds (ArrayLike) ‚Äì Predictions as returned by a classifier.

  • num_classes (int) ‚Äì Number of classes.

  • top_k (Optional[int]) ‚Äì If not None, calculate precision only on the top k highest probability predictions. Default is None. If None, calculate precision on all predictions.

  • average (Literal["micro", "macro", "weighted", None]) ‚Äì

    Average to apply. If None, return scores for each class. Default is None. One of:

    • "micro": Calculate metrics globally by counting the total true

      positives, false negatives and false positives.

    • "macro": Calculate metrics for each label, and find their

      unweighted mean. This does not take label imbalance into account.

    • "weighted": Calculate metrics for each label, and find their

      average weighted by support (the number of true instances for each label). This alters ‚Äúmacro‚Äù to account for label imbalance.

    Note that ‚Äúsamples‚Äù is not supported. Use multilabel_precision instead.

  • sample_weight (Optional[ArrayLike]) ‚Äì Sample weights.

  • zero_division (Literal["warn", 0, 1]) ‚Äì Value to return when there are no true positives or true negatives. If set to ‚Äúwarn‚Äù, this acts as 0, but warnings are also raised.

Returns

precision

Return type
