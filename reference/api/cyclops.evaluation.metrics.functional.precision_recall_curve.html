binary_precision_recall_curve(target, preds, thresholds=None, pos_label=1)[source]ÔÉÅ

Compute precision-recall curve for binary input.

Parameters

  • target (ArrayLike) ‚Äì Binary target values.

  • preds (ArrayLike) ‚Äì Predicted probabilities or output of a decision function. If preds is not in the [0, 1] range, it will be transformed into this range via the sigmoid function.

  • thresholds (int or list of floats or np.ndarray of floats, default=None) ‚Äì Thresholds used for computing the precision and recall scores. If int, then the number of thresholds to use. If list or array, then the thresholds to use. If None, then the thresholds are automatically determined by the unique values in preds.

  • pos_label (int) ‚Äì The label of the positive class.

Return type

Tuple[ndarray, ndarray, ndarray]

Returns

  • precision (np.ndarray) ‚Äì Precision scores such that element i is the precision of predictions with score >= thresholds[i].

  • recall (np.ndarray) ‚Äì Recall scores in descending order.

  • thresholds (np.ndarray) ‚Äì Thresholds used for computing the precision and recall scores.

Examples

>>> from cyclops.evaluation.metrics.functional import (
...     binary_precision_recall_curve
... )
>>> target = [0, 0, 1, 1]
>>> preds = [0.1, 0.4, 0.35, 0.8]
>>> precision, recall, thresholds = binary_precision_recall_curve(target,
...     preds, thresholds=5
... )
>>> precision
array([0.5, 0.66666667, 1., 1., 0.]
>>> recall
array([1., 1., 0.5, 0.5, 0.])
>>> thresholds
array([0.1, 0.25 , 0.5, 0.75 , 1.])

Compute the precision-recall curve for multilabel input.

Parameters

  • target (ArrayLike) ‚Äì The target values.

  • preds (ArrayLike) ‚Äì Predicted probabilities or output of a decision function. If the values are not in [0, 1], then they are converted into that range by applying the sigmoid function.

  • num_labels (int) ‚Äì Number of labels.

  • thresholds (np.ndarray) ‚Äì Thresholds used for computing the precision and recall scores. If int, then the number of thresholds to use. If list of floats, then the thresholds to use. If None, then the thresholds are computed automatically from the unique values in preds.

Return type

Union[Tuple[ndarray, ndarray, ndarray], Tuple[List[ndarray], List[ndarray], List[ndarray]]]

Returns

  • precision (np.ndarray or List[np.ndarray]) ‚Äì Precision values for each label. If thresholds is None, then precision is a list of arrays, one for each label. Otherwise, precision is a single array with shape (num_labels, len(thresholds)).

  • recall (np.ndarray or List[np.ndarray]) ‚Äì Recall values for each label. If thresholds is None, then recall is a list of arrays, one for each label. Otherwise, recall is a single array with shape (num_labels, len(thresholds)).

  • thresholds (np.ndarray or List[np.ndarray]) ‚Äì If thresholds is None, then thresholds is a list of arrays, one for each label. Otherwise, thresholds is a single array with shape (len(thresholds,).

Examples

>>> from cyclops.evaluation.metrics.functional import (
...     multilabel_precision_recall_curve)
>>> target = [[1, 1, 0], [0, 1, 0]]
>>> preds = [[0.1, 0.9, 0.8], [0.05, 0.95, 0.35]]
>>> precision, recall, thresholds = multilabel_precision_recall_curve(
...     target, preds, num_labels=3, thresholds=5)
>>> precision

Compute the precision-recall curve for different tasks/input types.

Parameters

  • target (ArrayLike) ‚Äì Ground truth (correct) target values.

  • preds (ArrayLike) ‚Äì Estimated probabilities or non-thresholded output of decision function.

  • task (Literal["binary", "multiclass", "multilabel"]) ‚Äì The type of task for the input data. One of ‚Äòbinary‚Äô, ‚Äòmulticlass‚Äô or ‚Äòmultilabel‚Äô.

  • thresholds (int or list of floats or np.ndarray of floats, default=None) ‚Äì Thresholds used for computing the precision and recall scores. If int, then the number of thresholds to use. If list or array, then the thresholds to use. If None, then the thresholds are automatically determined by the sunique values in preds

  • pos_label (int, default=1) ‚Äì The label of the positive class.

  • num_classes (int, optional) ‚Äì The number of classes in the dataset. Required for multiclass tasks.

  • num_labels (int, optional) ‚Äì The number of labels in the dataset. Required for multilabel tasks.

Return type

Union[Tuple[ndarray, ndarray, ndarray], Tuple[List[ndarray], List[ndarray], List[ndarray]]]

Returns

  • precision (np.ndarray) ‚Äì The precision scores where precision[i] is the precision score for scores >= thresholds[i]. If task is ‚Äòmulticlass‚Äô or ‚Äòmultilaabel‚Äô, then precision is a list of numpy arrays, where precision[i] is the precision scores for class or label i.

  • recall (np.ndarray) ‚Äì The recall scores where recall[i] is the recall score for scores >= thresholds[i]. If task is ‚Äòmulticlass‚Äô or ‚Äòmultilaabel‚Äô, then recall is a list of numpy arrays, where recall[i] is the recall scores for class or label i.

  • thresholds (np.ndarray) ‚Äì Thresholds used for computing the precision and recall scores.

Raises

  • ValueError ‚Äì If task is not one of ‚Äòbinary‚Äô, ‚Äòmulticlass‚Äô or ‚Äòmultilabel‚Äô.

  • AssertionError ‚Äì If task is multiclass and num_classes is not provided.

  • AssertionError ‚Äì If task is multilabel and num_labels is not provided.
