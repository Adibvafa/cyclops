cyclops.evaluation.metrics.functional.stat_scores moduleÔÉÅ

Functions for computing stat scores for different types of inputs.

The stat scores are the number of true positives, false positives, true negatives, and false negatives. Binary, multiclass and multilabel data are supported, including logits and probabilities.

ndarray

  • The number of true positives, false positives, true negatives and false

  • negatives and the support.
