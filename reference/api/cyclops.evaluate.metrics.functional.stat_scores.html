cyclops.evaluate.metrics.functional.stat_scores moduleÔÉÅ

Functions for computing stat scores for different types of inputs.

The stat scores are the number of true positives, false positives, true negatives, and false negatives. Binary, multiclass and multilabel data are supported, including logits and probabilities.

binary_stat_scores(target, preds, pos_label=1, threshold=0.5)[source]ÔÉÅ

Compute the stat scores for binary inputs.

Parameters

  • target (ArrayLike) ‚Äì Ground truth.

  • preds (ArrayLike) ‚Äì Predictions.

  • pos_label (int, default=1) ‚Äì The label to use for the positive class.

  • threshold (float, default=0.5) ‚Äì The threshold to use for converting the predictions to binary values. Logits will be converted to probabilities using the sigmoid function.

Returns

The true positives, false positives, true negatives and false negatives and support in that order.

Return type

numpy.ndarray

Raises

ValueError ‚Äì If the threshold is not in [0, 1] or if the pos_label is not 0 or 1.

Examples

>>> from cyclops.evaluation.metrics.functional import binary_stat_scores
>>> target = [0, 1, 1, 0]
>>> preds = [0, 1, 0, 0]
>>> binary_stat_scores(target, preds)
array([1, 0, 2, 1, 2])

Compute stat scores for multiclass targets.

Parameters

  • target (ArrayLike) ‚Äì The ground truth values.

  • preds (ArrayLike) ‚Äì The predictions. If determined to be in continuous format, will be converted to multiclass using the top_k parameter.

  • num_classes (int) ‚Äì The total number of classes for the problem.

  • top_k (Optional[int], default=None) ‚Äì The number of top predictions to consider when computing the stat scores. If None, it is assumed to be 1.

  • classwise (bool, default=True) ‚Äì Whether to return the stat scores for each class or sum over all classes.

Returns

The number of true positives, false positives, true negatives, false negatives and support. If classwise is True, the shape is (num_classes, 5). Otherwise, the shape is (5,)

Return type

numpy.nadarray

Examples

>>> from cyclops.evaluation.metrics.functional import multiclass_stat_scores
>>> target = [0, 1, 2, 2, 2]
>>> preds = [0, 2, 1, 2, 0]
>>> multiclass_stat_scores(target, preds, num_classes=3)
array([[1, 1, 3, 0, 1],
       [0, 1, 3, 1, 1],
       [1, 1, 1, 2, 3]])

Compute the stat scores for multilabel inputs.

Parameters

  • target (ArrayLike) ‚Äì Ground truth.

  • preds (ArrayLike) ‚Äì Predictions.

  • num_labels (int) ‚Äì The total number of labels for the problem.

  • threshold (float, default=0.5) ‚Äì Threshold value for binarizing predictions that are probabilities or logits. A sigmoid function is applied if the predictions are logits.

  • top_k (int, default=None) ‚Äì The number of top predictions to consider when computing the statistics.

  • labelwise (bool, default=False) ‚Äì Whether to return the stat scores for each label or sum over all labels.

The number of true positives, false positives, true negatives and false negatives and the support. The shape of the array is (5, num_labels) if labelwise=True and (5,) otherwise.

numpy.ndarray

ValueError ‚Äì If threshold is not between 0 and 1.

stat_scores(target, preds, task, pos_label=1, threshold=0.5, num_classes=None, classwise=True, top_k=None, num_labels=None, labelwise=False)[source]ÔÉÅ

Compute stat scores for binary, multiclass or multilabel problems.

This function acts as an entry point to the specialized functions for each task.

Parameters

  • target (ArrayLike) ‚Äì Ground truth.

  • preds (ArrayLike) ‚Äì Predictions.

  • task (Literal["binary", "multiclass", "multilabel"]) ‚Äì The task type. Can be either binary, multiclass or multilabel.

  • pos_label (int, default=1) ‚Äì The positive label to report. Only used for binary tasks.

  • threshold (float, default=0.5) ‚Äì The threshold to use for binarizing the predictions if logits or probabilities are provided. If logits are provided, a sigmoid function is applied prior to binarization. Used for binary and multilabel tasks.

  • num_classes (int) ‚Äì The number of classes for the problem. Required for multiclass tasks.

  • classwise (bool, default=True) ‚Äì Whether to return the stat scores for each class or sum over all classes. Only used for multiclass tasks.

  • top_k (int, default=None) ‚Äì The number of top predictions to consider when computing the statistics. If None, top_k is set to 1. Used for multiclass and multilabel tasks.

  • num_labels (int) ‚Äì The number of labels. Only used for multilabel tasks.

  • labelwise (bool, default=False) ‚Äì Whether to compute the stat scores labelwise. Only used for multilabel tasks.

Returns

scores ‚Äì The stat scores - true positives, false positives, true negatives, false negatives and support. For binary tasks, the shape is (5,). For multiclass tasks, the shape is (n_classes, 5) if classwise is True, otherwise (5,). For multilabel tasks, the shape is (n_labels, 5) if labelwise is True, otherwise (n_classes, 5).

Return type

numpy.ndarray
