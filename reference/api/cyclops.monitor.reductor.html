Reductor Module.

class Reductor(dr_method, model_path=None, var_ret=0.8, n_components=None, n_features=None, n_classes=None, random_state=42)[source]ÔÉÅ

Bases: object

The Reductor class is used to reduce the dimensionality of the data.

The reductor is initialized with a dimensionality reduction method. The reductor can then be fit to the data and used to transform the data.

>>> from drift_detection.reductor import Reductor
>>> from sklearn.datasets import load_diabetes
>>> X, y = load_diabetes(return_X_y=True)
>>> reductor = Reductor("PCA")
>>> reductor.fit(X)
>>> X_transformed = reductor.transform(X)
