Bases: object

An interface dataclass to wrap queries, and run them with post-processing.

An similar dataclass to QueryInterface, where custom post-processing functions on the pandas dataframe returned from the query can be run. However, this prevents the query from being further used, and hence is declared as private attribute.

Parameters

  • database (cyclops.orm.Database) ‚Äì Database object to create ORM, and query data.

  • _query (cyclops.query.util.TableTypes) ‚Äì The query.
