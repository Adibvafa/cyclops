accuracy(target, preds, task, pos_label=1, num_classes=None, threshold=0.5, top_k=None, num_labels=None, average=None, sample_weight=None, zero_division='warn')[source]ÔÉÅ

Compute accuracy score for different classification tasks.

Parameters

  • target (ArrayLike) ‚Äì Ground truth (correct) target values.

  • preds (ArrayLike) ‚Äì Estimated targets (predictions) as returned by a classifier.

  • task (Literal["binary", "multiclass", "multilabel"]) ‚Äì The type of task for the input data. One of ‚Äòbinary‚Äô, ‚Äòmulticlass‚Äô or ‚Äòmultilabel‚Äô.

  • pos_label (int, default=1) ‚Äì The class to report. Can be 0 or 1.

  • num_classes (int, optional) ‚Äì The number of classes in the dataset. Required for multiclass tasks.

  • threshold (float, default=0.5) ‚Äì The threshold value for converting probability or logit scores to binary. A sigmoid function is first applied to logits to convert them to probabilities.

  • top_k (int, default=None) ‚Äì The number of top predictions to consider when computing accuracy. Required for multiclass and multilabel tasks.

  • num_labels (int, optional) ‚Äì The number of labels in the dataset. Required for multilabel tasks.

  • average (Literal["micro", "macro", "weighted", "samples", None], default=None) ‚Äì

    The type of averaging to apply to the accuracy scores. One of:

    • micro: Calculate metrics globally by counting the total true positives, false negatives and false positives.

    • macro: Calculate metrics for each class, and find their unweighted mean. This does not take class imbalance into account.

    • weighted: Calculate metrics for each class, and find their average, weighted by support (the number of true instances for each class). This alters macro to account for class imbalance.

    • samples: Calculate metrics for each instance, and find their average (only meaningful for multilabel classification).

    • None: The scores for each class or label are returned.

  • sample_weight (ArrayLike, default=None) ‚Äì Sample weights.

  • zero_division (Literal["warn", 0, 1], default="warn") ‚Äì Sets the value to return when there is a zero division. If set to warn, this acts as 0, but warnings are also raised.

Returns

accuracy_score ‚Äì The average accuracy score as a float if average is not None, otherwise a numpy array of accuracy scores per class/label.

Return type

Union[float, np.ndarray]

Raises

  • ValueError ‚Äì If task is not one of binary, multiclass or multilabel.

  • AssertionError ‚Äì If task is multiclass and num_classes is not provided or is less than 0.

  • AssertionError ‚Äì If task is multilabel and num_labels is not provided or is less than 0.

Examples

>>> from cyclops.evaluation.metrics.functional import accuracy
>>> target = [0, 1, 0, 1]
>>> preds = [0, 1, 1, 1]
>>> accuracy(target, preds, task="binary")
0.75

>>> target = [0, 1, 2, 2, 2]
>>> preds = [0, 0, 2, 2, 1]
>>> accuracy(target, preds, task="multiclass", num_classes=3, average="micro")
0.6

>>> target = np.array([[0, 1, 1], [1, 0, 0]])
>>> preds = np.array([[0, 1, 0], [1, 0, 1]])
>>> accuracy(target, preds, task="multilabel", num_labels=3, average="mcro")
0.6666666666666666

Compute the accuracy score for multiclass classification problems.

Parameters

  • target (ArrayLike) ‚Äì Ground truth (correct) target values.

  • preds (ArrayLike) ‚Äì Estimated targets (predictions) as returned by a classifier.

  • num_classes (int) ‚Äì Number of classes in the dataset.

  • top_k (int, default=None) ‚Äì Number of highest probability predictions or logits to consider when computing the accuracy score.

  • sample_weight (ArrayLike, default=None) ‚Äì Sample weights.

  • average (Literal["micro", "macro", "weighted", None], default=None) ‚Äì

    If not None, this determines the type of averaging performed on the data:

    • micro: Calculate metrics globally by counting the total true positives, false negatives and false positives.

    • macro: Calculate metrics for each class, and find their unweighted mean. This does not take class imbalance into account.

    • weighted: Calculate metrics for each class, and find their average, weighted by support (the number of true instances for each class). This alters macro to account for class imbalance.

  • zero_division (Literal["warn", 0, 1], default="warn") ‚Äì Sets the value to return when there is a zero division. If set to warn, this acts as 0, but warnings are also raised.

Returns

The average accuracy score as a float if average is not None, otherwise a numpy array of accuracy scores per class/label.

Return type

float or np.ndarray

Raises

ValueError ‚Äì If average is not one of micro, macro, weighted or None.

Examples

>>> from cyclops.evaluation.metrics.functional import multiclass_accuracy
>>> target = [0, 1, 2, 2, 2]
>>> preds = [0, 0, 2, 2, 1]
>>> multiclass_accuracy(target, preds, num_classes=3)
array([1.        , 0.        , 0.66666667])

>>> multiclass_accuracy(target, preds, num_classes=3, average="micro")
0.6

>>> multiclass_accuracy(target, preds, num_classes=3, average="macro")
0.5555555555555555

>>> multiclass_accuracy(target, preds, num_classes=3, average="weighted")
0.6

Compute the accuracy score for multilabel-indicator input.

Parameters

  • target (array-like of shape (num_samples, num_labels)) ‚Äì Ground truth (correct) target values.

  • preds (array-like of shape (num_samples, num_labels)) ‚Äì Estimated targets as returned by a classifier.

  • num_labels (int) ‚Äì Number of labels in the multilabel classification task.

  • threshold (float, default=0.5) ‚Äì Threshold value for binarizing the output of the classifier.

  • top_k (int, optional, default=None) ‚Äì The number of highest probability or logit predictions considered to find the correct label. Only works when preds contains probabilities/logits.

  • sample_weight (array-like of shape (num_samples,), default=None) ‚Äì Sample weights.

  • average (Literal['micro', 'macro', 'samples', 'weighted', None], default=None) ‚Äì

    If None, return the accuracy score per label, otherwise this determines the type of averaging performed on the data:

    • micro: Calculate metrics globally by counting the total true positives, false negatives and false positives.

    • macro: Calculate metrics for each label, and find their unweighted mean. This does not take label imbalance into account.

    • weighted: Calculate metrics for each label, and find their average, weighted by support (the number of true instances for each label).

    • samples: Calculate metrics for each instance, and find their average.

  • zero_division (Literal['warn', 0, 1], default="warn") ‚Äì Sets the value to return when there is a zero division. If set to warn, this acts as 0, but warnings are also raised.

Returns

The average accuracy score as a flot if average is not None, otherwise a numpy array of accuracy scores per label.

Return type

float or np.ndarray

Raises

ValueError ‚Äì If average is not one of micro, macro, weighted, samples, or None.

Examples

>>> import numpy as np
>>> from cyclops.evaluation.metrics.functional import multilabel_accuracy
>>> target = np.array([[0, 1, 1], [1, 0, 0]])
>>> preds = np.array([[0, 1, 0], [1, 0, 1]])
>>> multilabel_accuracy(target, preds, num_labels=3, average=None)
array([1., 1., 0.])

>>> multilabel_accuracy(target, preds, num_labels=3, average="micro")
0.6666666666666666
